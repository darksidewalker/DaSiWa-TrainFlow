Windows updates `python_embeded/windows` with the official Python 3.12 embeddable package. Linux creates `python_embeded/linux` as a local venv using `python3.12` or `python3` from PATH. Dependencies install with uv first and fall back to pip if needed. Backups are deleted after a successful update unless the checkbox is enabled. Flash Attention is skipped unless selected, searches official and community prebuilt wheels, CUDA 13.0 uses the default torch/SDPA attention path when no compatible wheel exists, and local flash-attn source builds are guarded because they can exhaust RAM and swap.
